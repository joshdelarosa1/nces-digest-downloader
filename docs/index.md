# NCES Digest Downloader

NCES Digest Downloader automates retrieval of NCES Digest Excel tables with resilient scraping,
filtering, and logging workflows.

## Project Scope

This tool focuses on:

- Discovering digest table pages by year
- Resolving downloadable Excel links
- Downloading artifacts with retry/throttle safeguards
- Capturing hash and metadata logs for reproducible workflows
